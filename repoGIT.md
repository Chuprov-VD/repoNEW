# Инструкция для работы с Markdown

## Выделение текста
 * Чтобы выделить текст курсивом нужно, обромить зведочками (*) Например: *вот так*
* Чтобы выделить текст полужирным нужно, обромить двойными зведочками (**) Например: **вот так**
